## Installation

```
  git clone https://github.com/yarokon/users-manager.git
  cd users-manager
  touch .env
    PORT=4000
    MONGODB_URI=<Paste connection string>
  yarn
  cd client && yarn
```

## Run Project Locally

```
  terminal 1: yarn dev
  terminal 2: cd cliend && yarn start
```
